# 滑动验证。  
源代码链接：[http://www.xttblog.com/?p=1128](http://www.xttblog.com/?p=1128)，对作者示例代码做了包装和优化。将css整合到了js中，否则短短几行css样式还要单独占用一个css文件。

# 依赖

* jquery 1.x
* html5

``` js
  var drager = $('#dragDemo').drag();
```
